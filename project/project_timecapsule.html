Time Capsule

YUKI yukimew.github.io/interactive1 ONLINE ZINE 2019

This is a time capsule recording the changes for online applications and platforms between 2005 and 2019.

◄Youku is the first video application I learned to use online. I really like ACG (animation, comic and game),but at that time, China did not exist any application for watching ACG so I just use simple video platform until Bilibili occured.

►Bilibili is a video platform which was created in 2012 also helpful for me to search and find animation, as well as game. Also,it a platform where users can submit, view, and add commentary subtitles on videos.

◄QQ is the first application for chatting which is intronduced by the online game friends to me. Then, I still love using this application for making online friends, it is also convinience for me to send document, sharing motion on Qzone and chat in groups.

►Wechat is a social media also created by Tencent in 2011. Because of its simple and clean interface， it‘s so easy for older people who never use application on phone start to learn how to use, so I use wechat to communication with my family a lot but not using QQ.

◄ChuanQi is the first way that I can communicate with strangers online. It is a 2D MMORPG online game created by Korean and was introduce in China.

►Final Fantasy XIV also is a kind of MMORPG online game but it is 3D and made by Japan. It is my favourite online game now. It is introduced into China, Korean, North American and Europe.
